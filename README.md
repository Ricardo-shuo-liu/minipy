## Description
A lightweight interpreter fully handwritten in standard C language, built without any third-party parsing tools like Flex or Bison. This project implements the complete compilation workflow from source code scanning to virtual machine execution, with lexical analysis, syntax parsing, AST construction, bytecode generation and a custom stack virtual machine.

## Project Overview
MiniPy is an educational interpreter designed to demonstrate the full pipeline of classic compilation principles. Every core module is coded manually from scratch. It can parse custom script files ending with `.mp`, execute arithmetic and logical logic, and output results through the built-in print statement. The program provides precise line number prompts when lexical or syntax errors occur.

## Core Functional Modules
1. Lexical Analyzer (Lexer)
- Automatically ignore spaces, tabs, blank lines and line breaks
- Support single-line comment starting with `#`
- Recognize numbers, identifiers and reserved keywords
- Identify all arithmetic and comparison operators: `+ - * / = == != < > <= >=`
- Capture illegal characters and report errors with specific line numbers

2. Syntax Analyzer (Parser)
- Recursive descent parser implemented manually
- Support arithmetic expressions and comparison expressions
- Parse variable assignment statements
- Implement `if-else` conditional branches and `while` loop structures
- Detect missing expressions and syntax mismatches

3. Abstract Syntax Tree (AST)
Construct tree nodes to store parsed semantic information, separating parsing logic and intermediate code generation.

4. Bytecode Generator
Traverse AST nodes and convert high-level syntax into dedicated bytecode instructions for virtual machine.

5. Stack-based Virtual Machine (VM)
Execute generated bytecode sequentially and offer built-in print function for numeric output.

## Supported Syntax Demo
### Variable Assignment
```
a = 123;
b = a + 456;
```

### Arithmetic Calculation
```
res = 10 + 20 * 3 - 15 / 5;
```
### Comparison Operators
`== != < > <= >=`
```
flag = i <= 10;
```
### If-Else Branch
```
if (a > 0) {
print(a);
} else {
print(0);
}
```
### While Loop
```
i = 0;
while (i < 5) {
print(i);
i = i + 1;
}
```
### Comment
```
full line comment
num = 777; # trailing comment
```

## Compile & Run Commands
Clear compiled files
```
make clean
```
Compile the whole project
```
make
```
Run single script
```
./minipy scripts/test_basic.mp
```
Run all test scripts
```
make test_all
```
Project Features

    No third-party dependencies, all components implemented manually
    Clear modular division following standard compilation process
    Accurate line-number error hints for debugging
    Complete closed-loop implementation: source text → token → AST → bytecode → runtime output
