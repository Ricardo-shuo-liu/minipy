## 简介
一款使用标准C语言完全手写的轻量级解释器，未使用Flex、Bison等任何第三方语法解析工具。本项目实现了从源码扫描到虚拟机执行的完整编译流程，包含词法分析、语法解析、抽象语法树构建、字节码生成以及自定义栈式虚拟机。

## 项目概述
MiniPy 是一款教学向解释器，完整展示经典编译原理全链路流程。所有核心模块均从零手动编写。程序可解析后缀为 `.mp` 的自定义脚本文件，执行算术与比较逻辑，并通过内置 print 语句输出运行结果。当出现词法或语法错误时，程序会精准标注错误所在行号。

## 核心功能模块
1. 词法分析器（Lexer）
- 自动忽略空格、制表符、空行与换行符
- 支持以 `#` 开头的单行注释
- 识别数字、标识符与系统保留关键字
- 完整识别所有算术、比较运算符：`+ - * / = == != < > <= >=`
- 捕获非法字符，并附带精确行号报错

2. 语法分析器（Parser）
- 手工实现递归下降语法分析器
- 支持算术表达式、比较表达式解析
- 解析变量赋值语句
- 实现 `if-else` 条件分支、`while` 循环两种控制结构
- 检测表达式缺失、语法符号不匹配等问题

3. 抽象语法树（AST）
构建树形节点存储解析后的语义信息，将语法解析逻辑与中间代码生成逻辑解耦。

4. 字节码生成器
遍历抽象语法树节点，将高层语法转换为虚拟机专用字节码指令。

5. 栈式虚拟机（VM）
按顺序执行生成的字节码，内置 print 函数用于数值打印输出。

## 项目目录结构
```
minipy/
├── include/        # 头文件存放目录
├── src/            # 源代码目录
│ ├── lexer.c
│ ├── parser.c
│ ├── ast.c
│ ├── codegen.c
│ ├── vm.c
│ ├── utils.c
│ └── main.c
├── scripts/        # .mp 测试脚本目录
├── obj/            # 编译生成的目标文件
└── Makefile        # 编译构建脚本
```

## 支持语法示例
### 变量赋值
```
a = 123;
b = a + 456;
```

### 四则运算
```
res = 10 + 20 * 3 - 15 / 5;
```
### 比较运算符
可用运算符：`== != < > <= >=`
```
flag = i <= 10;
```
### If-Else 条件分支
```
if (a > 0) {
    print(a);
} else {
    print(0);
}
```
### While 循环
```
i = 0;
while (i < 5) {
    print(i);
    i = i + 1;
}
```
### 注释写法
```
# 整行注释
num = 777; # 行尾注释
```

## 编译与运行命令
清理编译产物
```
make clean
```
完整编译项目
```
make
```
运行单个脚本
```
./minipy scripts/test_basic.mp
```
批量运行全部测试脚本
```
make test_all
```

## 项目特点
- 无任何第三方依赖，全部组件手写实现
- 模块划分清晰，严格遵循标准编译流程
- 报错附带精准行号，便于调试
- 完整闭环实现：源码文本 → 词法记号 → 抽象语法树 → 字节码 → 虚拟机运行输出
